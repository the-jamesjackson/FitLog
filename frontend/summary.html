FitLog Project Summary

Topic & Purpose

FitLog is a fitness tracking website that allows users to log workouts and set fitness goals.

How It Works

  • No authentication
  • Users can log workouts with exercise type, date, duration, and notes
  • Users can create and complete fitness goals

Source of Data

Workout and goal data is stored in a PostgreSQL database

Database Diagram

Features Implemented

  • Workout log form and list with DB storage
  • Goal tracking with completion button
  • Dynamic form population from backend

API / Libraries Used

  • No third-party API
  • React
  • Bootstrap

CSS Frameworks

Bootstrap 5.3 was used for styling
